# Amsterdam Studio — Portafolio Web

**Desarrollador Front-End:** Russo  
Creativo y especializado en interfaces limpias, responsivas y con personalidad.

---

## Descripción

Este proyecto es la web oficial de Amsterdam Studio, un estudio creativo especializado en diseño gráfico aplicado. Aquí podrás explorar servicios, trabajos realizados y formas de contacto, todo diseñado con una fuerte narrativa visual y modularidad técnica.

Preview: https://amsterdamstudio.netlify.app/

---

## Tecnologías y Herramientas

- React con Vite.js para un desarrollo rápido y moderno
- Tailwind CSS para estilos responsivos y personalizados
- Componentes visuales externos (React Bits, ShadCN UI) para mejorar la experiencia UI
- Configuración de alias para mantener la arquitectura limpia y modular
- Animaciones y transiciones suaves para una experiencia envolvente

---

## Funcionalidades destacadas

- Navbar responsive y accesible con menú hamburguesa para móviles
- Hero con diseño oscuro y tipografía jerárquica
- Sección About con imagen de fondo y alineación a la derecha para una narrativa impactante
- Galería interactiva tipo slider con descripción y testimonios
- Sección Servicios con íconos y tarjetas animadas
- Contacto estilizado con enlaces directos y botones CTA
